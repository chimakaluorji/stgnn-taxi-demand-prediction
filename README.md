# Spatiotemporal Graph Neural Network for Taxi Demand Prediction

This repository contains the **frontend and backend implementation** of a
Spatiotemporal Graph Neural Network (ST-GNN) system for predicting high-demand taxi zones.

## Project Structure

- `frontend/` – ReactJS admin dashboard
- `backend/` – FastAPI + ST-GNN backend
